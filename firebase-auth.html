🔥 Firebase WeChat Login (Free Solution)

📋 Setup Steps

Step 1: Create Firebase Project

Go to Firebase Console

1. Click "Add project"
2. Enter project name: "chinese-heritage"
3. Enable Google Analytics (optional)
4. Click "Create project"

Step 2: Enable Authentication

In Firebase Console → Authentication → Sign-in method

1. Click "Get started"
2. Enable "WeChat" provider
3. Download config file
4. Add your WeChat AppID (if you have one)

Step 3: Get Firebase Config

Project Settings → General → Your apps → Web app → Firebase SDK snippet

const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  appId: "your-app-id"
};

Step 4: Add to Your Website

Add Firebase SDK to your HTML files

<script src="https://www.gstatic.com/firebasejs/9.6.1/firebase-app-compat.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.6.1/firebase-auth-compat.js"></script>

🚀 Quick Implementation

Replace your current WeChat login function with this:

// Initialize Firebase
const firebaseConfig = {
  apiKey: "your-api-key",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id"
};
firebase.initializeApp(firebaseConfig);
const auth = firebase.auth();

// WeChat login function
async function signInWithWeChat() {
  try {
    const provider = new firebase.auth.WechatAuthProvider();
    await auth.signInWithRedirect(provider);
  } catch (error) {
    console.error('WeChat login error:', error);
  }
}
